    支付方式

    货到付款：
    由MCAKE外送人员全程配送蛋糕，您可以在收到货物时，再进行订购商品的结算，可支付现金或使用POS机刷卡。

     

    网上支付：
    用户订购蛋糕，在结算页可以直接使用网上支付方式进行付款，网上支付方式主要包括：支付宝、网银在线支付。
    1、支付宝支付
    支付宝，全球领先的独立第三方支付平台。
    "支付宝账户"就是网上购物付款的"电子钱包"，将钱充入，随买随付，方便安全。
    2、网银在线支付


    领先的第三方电子支付专家。
    支持多家网银在线支付，同时涵盖工行、招行、建行、交行、农行、银联等国内20余种银行卡。

     

    预付费卡支付：
    订购结算页，也可以勾选使用预付费卡支付方式，目前支持得仕通、乐支付、付费通等10余种预付费卡，用户可以选择直接线上支付，也可以选择线下支付。

     

    异地付款：
    用户为朋友订购商品，如果选择异地付款方式，则配送员在配送蛋糕完成后，将根据用户指定地点，进行订购商品的结算，用户可以选择使用现金支付或POS刷卡。
    温馨提示：异地付款将会产生额外的服务费用【收款地点为外环以内及莘庄地区的，收取10元服务费；收款地点为外环以外的，客服会联系您加收服务费（郊县暂不提供配送、收款等服务）；具体配送范围标准结算页有图示展示】建议使用在线支付方式。

     


    优惠方式

    现金卡：
    现金卡是MCAKE推出的一种优惠卡，主要在线下推广方式中进行售卖。用户官网购买商品，在结算页勾选使用现金卡选项，输入现金卡号，即可进行相应金额抵扣。


    优惠券：
    优惠券发放是官网不定时进行优惠的一种活动形式。优惠券一般在官网活动中发放，也有些直接加入到用户账户内，在订单结算页，可以直接勾选使用优惠券选项，进行相应抵扣。


    温馨提示：随着MCAKE官网不断更新，可能还会有其他优惠方式出现，用户可以实时关注"会员中心"进行查看。
