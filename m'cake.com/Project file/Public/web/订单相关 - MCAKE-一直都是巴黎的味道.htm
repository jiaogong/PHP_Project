    订单查询

    登录MCAKE官网，点击首页上方导航栏"会员中心" ，在左侧"我的订单"中，用户可以查询在MCAKE的所有订单，包括已完成、已取消、待付款、待收货、待评价订单等。

     


    取消订单

    1、"线上支付方式"提交订单，但未进行付款：进入"会员中心 — 我的订单"，查找刚刚的订单，点击【取消订单】按钮取消订单。
    2、"线上支付"成功或选择"货到付款方式"已审核通过：致电客服热点：4006—678—678进行订单状态的修改。
