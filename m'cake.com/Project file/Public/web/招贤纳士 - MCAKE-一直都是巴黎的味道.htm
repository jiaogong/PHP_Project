    移动端APP开发工程师


    【岗位职责】

    1.   负责在移动端平台下的应用程序设计、开发、发布和维护；

    2.   负责移动端平台项目的架构设计、方案的制定；

    3.   跟进移动端平台的新技术发展，编写设计开发及实现文档；

    4.   优化移动产品的质量、性能及用户体验。

     

    【任职要求】

    1.   本科以上学历，3年以上互联网从业经验，精通手机客户端开发平台，至少2年以上的实际iPhone及Android平台开发经验；

    2.   具有电商类产品App store上线应用经验者优先；

    3.   精通Android/iOS工作机制和内核，精通C/C++，并对其他开发语言有所了解；

    4.   精通移动互联网应用协议，精通TCP/IP协议与编程；

    5.   有解决问题、钻研新技术的兴趣和能力,善于交流和表达,有良好的团队合作精神；

    6.   有良好的分析问题和解决问题的能力，良好的工作进度管理与把控能力，抗压能力强。

     


    前端开发工程师


    【岗位职责】

    1.   设计实现数据产品的用户界面，优化前端性能，确保Web用户界面在多浏览器环境中正常工作； 

    2.   负责从设计图制作HTML，包括CSS，JavaScript等； 

    3.   JS前端页面交互设计，提高用户体验；

    4.   理解后端架构，与后端工程师配合，为项目提供最优化的技术解决方案。 

     

    【任职要求】

    1.   大学本科或以上学历，计算机等相关专业，有两年以上互联网产品前端开发经验； 

    2.   有H5页面开发经验，熟悉移动端适配优先；

    3.   熟悉原生 javascript，熟悉 jQuery/Zepto框架

    4.   精通至少一种前端MV*类框架，如：Backbone.js、AngularJS、Ember.js等；

    5.   熟悉Android和iOS开发，熟悉使用PhoneGap/Cordova等前端框架开发Hybrid app；

    6.   有移动端电商或O2O项目经验者优先。具备良好的学习能力、沟通能力、分析及解决问题能力，优秀的团队协作精神。

    3.   对产品进行持续的优化，提升用户体验，深度挖掘用户需求；

    3.   结合数据分析，推动产品进行快速叠带更新；

    2.   深刻理解互联网产品管理，理解互联网思维； 

    3.   计算机相关专业，5年以上互联网工作经验，2年以上移动互联网工作经验，有知名移动产品经验者优先考虑；

    4.   熟悉电商营销相关业务细节、流程； 熟悉数据分析、数据挖掘的方法和理念；

    2.  负责品牌形象等宣传资料、广告资料等相关内容的设计；

    3.  推广活动的专题设计，及品牌推广广告的创意设计；

    4.  组织设计师全面把握重要项目的配套设计；

    5.  摄影方案构思和创意。

     

    【任职要求】

    1.  大专以上学历，美术、设计类科班专业，良好的设计功底和创意及审美能力，2年以上相关经验；

    2.  爱艺术，爱生活，爱设计，有决心将设计做好尽善尽美；

    3.  具备互联网思维及热爱设计的达人，喜欢琢磨市场趋势、用户使用习惯、爱摸索探寻分析用户需求；

    4.  敏锐的捕捉视觉流行趋势，把握品牌视觉风格，将自己独到的想法融入设计之中，能帮助提升团队的设计能力；

    5.  工作主动性高，善于沟通，能准确完整的表达自己的设计思路；有较强的责任心和执行能力。能独立完成设计及制作任务，具备良好团队合作精神和一定的团队管理能力；

    6.  4A广告公司或品牌电商相关工作经验优先考虑。
